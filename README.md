# UpbitHist2Tsv

업비트의 암호화폐 거래 이력을 엑셀에 쉽게 붙여넣을 수 있는 TSV(Tab-Separated Values) 형식으로 변환하는 프로그램입니다.

## 원본 형식
2019.10.26 08:47
	
BTC
	
KRW
	매도	
0.01442790BTC
	
10,030,000KRW
	
144,711KRW
...
